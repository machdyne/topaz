# Topaz Computer

Topaz is a headless computer that uses an ASIC module based on the open-source [KIAN-V RV32 SOC](https://github.com/splinedrive/gf180mcu-kianv-rv32ima-sv32).

![Topaz Computer](https://github.com/machdyne/topaz/blob/221c619fa4026424c389ca5b53fc48e792a5ca62/topaz.png)

This repo contains schematics, PCB layouts, firmware and documentation.
