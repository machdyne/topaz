# Topaz Computer

Topaz is a headless computer that uses an ASIC module based on the open-source [KIAN-V RV32 SOC](https://github.com/splinedrive/gf180mcu-kianv-rv32ima-sv32).

This repo contains schematics, PCB layouts, pinouts, firmware and documentation.
